Grade Manager

Created by: Jeffrey Tram

Click "Add Class" to get started! Then, add grades to each of your classes.

  • Enter the weight and grade as a percent
  • Switch between classes by clicking the class name in the bar
  • Change the class name by clicking the bolded name, and saving the name
  • For grades not received, leave the grade empty.
  • Save all other changes by clicking any button or switching between classes.
